# Roblox 5 Account AFK Bot

Script python ini buat login dan AFK di Roblox pakai 5 akun sekaligus.
Jalan bergiliran biar ringan untuk user gratisan replit.

## Fitur
- Login 5 akun otomatis
- Join game otomatis pakai Game ID
- Anti AFK tiap 1 menit
- Jalan 24 jam pakai UptimeRobot
- Pakai .env biar aman

## Cara Install

### 1. Upload file
Upload semua file ini ke Replit:
- main.py
- keep_alive.py
- accounts.py
- requirements.txt
- .env
- .gitignore

### 2. Install package
Buka tab Shell di replit, jalanin:
```bash
pip install -r requirements.txt
